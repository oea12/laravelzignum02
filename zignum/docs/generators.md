Thor\Generators
==========

Laravel 4 CRUD generators for Thor CMS models using Bootstrap 3

### Features
* Controllers, models, migrations and views generators for normal, translatable, pageable and imageable models.
* Generate multilingual-ready CRUD instantly with a single command  `thor:generate`
* Admin Resource Route Generator with automatic permission validation
* HTML generator and Bootstrap 3 Form helpers
